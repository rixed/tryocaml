Modules to hide implementation (2)

Of course you can't reach N2.one since N2 is a Number and Number have no such value. What robust would that be if a module which is only given a Number could try to guess members that are not in the interface?

So the lesson is: Two completely distinct implementations of a module type are interchangeable once they are forced into this module type. Other members these modules might have are hidden definitively and no program can tell the one from the other.
