Modules to hide type implementation (3)

In addition to hide definitions, applying a module signature to a module allows to hide a type definition while its declaration remain present, and thus usable in the following definitions.

Let's see an example:

module type Point = sig
  type t
  val origin : t
  val unit_x : t
  val unit_y : t
  val add : t -> t -> t
  val plot : t -> unit
end
